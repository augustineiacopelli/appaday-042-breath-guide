# 042 · Breath Guide

**AppADay** — Day 042

A meditative breathing guide with six researched breathing patterns, each with an animated glowing orb that expands and contracts with your breath cycle. A built-in calendar tracks every day you practice and maintains a running streak.

## Patterns included

- **Box Breathing** (4-4-4-4) — Equal inhale, hold, exhale, hold. Used for stress control and nervous system reset.
- **4-7-8 Breathing** — Dr. Andrew Weil’s relaxation technique; extended exhale activates the parasympathetic system.
- **Coherent Breathing** (5-5) — Five breaths per minute; maximizes heart rate variability.
- **Wim Hof Pattern** (2-0-2-1) — Energizing cyclic breath for increased oxygen.
- **Diaphragmatic** (4-0-6) — Deep belly breathing with a longer exhale; foundation of relaxation.
- **Contemplative** (3-3-3-3) — A gentle pattern suited to prayer, lectio divina, and silent sitting.

## Features

- Animated orb breathes with you using smooth easing
- Adjustable session rounds (1–20)
- Per-round progress dots
- Live phase label and countdown timer
- Practice calendar with month navigation, streak counter, and per-day history
- Star field background, no external dependencies beyond Google Fonts
